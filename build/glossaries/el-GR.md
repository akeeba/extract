# Glossary — el-GR (Ελληνικά)

Endonym: **Ελληνικά**. Auto option: **(Αυτόματο)**.

| English | Greek |
|---|---|
| archive (the backup archive) | αρχειοθήκη |
| archive file | αρχείο αρχειοθήκης |
| file | αρχείο |
| folder / directory | φάκελος |
| extract / extraction | εξαγωγή |
| output folder | φάκελος εξαγωγής |
| password | κωδικός πρόσβασης |
| decryption | αποκρυπτογράφηση |
| pattern | μοτίβο |
| warning | προειδοποίηση |
| corrupt | κατεστραμμένο |
| truncated | περικομμένο |
| part (of a multi-part archive) | τμήμα |
| backup | αντίγραφο ασφαλείας |
| header | κεφαλίδα |
| offset | μετατόπιση |
| Browse | Αναζήτηση |
| Cancel | Ακύρωση |
| Start | Έναρξη |
| Download | Λήψη |
| Insert | Εισαγωγή |
| drag and drop | μεταφορά και απόθεση |

Notes: use the Greek `·` (áno teleía), never `;`. "αρχειοθήκη" (archive) is kept
distinct from "αρχείο" (file), matching Greek archiver conventions (e.g. 7-Zip).
